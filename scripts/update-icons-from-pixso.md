# 从 Pixso 更新图标指南

## 步骤

1. **在 Pixso 中选中图标节点**
   - 打开设计稿
   - 选中要导出的图标
   - 复制链接（包含 item-id）

2. **导出图标**
   - 使用以下命令格式：
   ```
   提供图标节点 ID 或链接
   ```

3. **图标将自动保存到**
   - `assets/icons/icon/` - 功能图标（历史、设置、删除等）
   - `assets/icons/nav/` - 分类导航图标

## 需要替换的图标列表

### 功能图标
- history (历史)
- settings (设置)
- delete (删除)
- save-board (新建计划)
- save-image (保存图片)
- plus (放大)
- minus (缩小)
- location/layers (重置)

### 分类图标
- living (客厅)
- kitchen (厨房)
- bedroom (卧室)
- bath (浴室)
- dining (餐厅)
- office (办公室)
- kids (儿童房)
- study (书房)
- balcony (阳台)
- entrance (入口)
- corridor (走廊)

每个分类需要两个版本：
- default (默认状态)
- select (选中状态)
